# UX Spec: Star Map（星图）

**Screen ID**: `star_map`
**Type**: Core Gameplay Screen
**Input**: Mouse (click/select) + Keyboard (Space=end turn, Esc=cancel)

---

## Purpose

主游戏界面。玩家在此查看星图全局、选择星球、触发出征。是 90% 游戏时间的所在。

## Interaction Flow

### 主流程

```
点击星球 → 显示详情面板
    │
    ├── 己方星：高亮所有相邻敌星/中立星（金色边框）
    │       │
    │       └── 点击相邻敌星 → 弹出出征面板
    │              │
    │              ├── 确认发兵 → 面板关闭，部署命令提交
    │              └── 取消 (Esc) → 面板关闭
    │
    ├── 敌方/中立星：仅显示详情，不高亮相邻星
    │
    └── 再次点击同星 / 点击空白 → 取消选中
```

### 状态转换

| 当前状态 | 玩家操作 | 结果 |
|----------|----------|------|
| 无选中 | 点击己方星 | 选中，高亮相邻敌星，显示详情 |
| 无选中 | 点击敌方星 | 选中，显示详情，不触发操作 |
| 选中己方星 | 点击相邻敌星 | 弹出出征面板 |
| 选中己方星 | 点击非相邻敌星 | 切换选中到非相邻敌星 |
| 选中任一星 | 再次点击同星 | 取消选中 |
| 选中任一星 | 点击空白区域 | 取消选中 |
| 选中任一星 | 按 Esc | 取消选中 |
| 循环中 | 按 Space | 结束回合 |

## Visual Feedback

| 事件 | 视觉响应 | 时长 |
|------|----------|------|
| 选中星球 | 2px 白色边框 + 详情面板出现 | 即时 |
| 相邻敌星高亮 | 2px 金色边框 | 即时 |
| 兵力变化 | 数字直接跳变 | 即时 |
| 星球归属变更 | 颜色切换（红→蓝/蓝→红/灰→色） | 即时 |
| 回合结束 | 所有高亮取消 + 全图刷新 | 即时 |
| 游戏结束 | 半透明 overlay + "胜利"/"失败" | 淡入 300ms |

---

## EventBus Subscriptions

星图监听以下事件自动刷新（不轮询）：

| 信号 | 响应 |
|------|------|
| `planets_initialized` | 首次渲染 |
| `planet_owner_changed(id, old, new)` | 重绘该星球颜色 + 详情更新 |
| `planet_garrison_changed(id, old, new)` | 重绘兵力数字 + 详情更新 |
| `turn_ended(turn_number)` | 取消选中 + 全图重绘 |
| `game_ended(result)` | 显示胜利/失败 overlay |

---

## Edge Cases

- **快速双击星球**：第二次点击视为取消选中（如果已选中同一颗）
- **回合结算期间点击**：忽略 — 按钮在 EXECUTION/CLEANUP 阶段禁用
- **空星图（0 星球）**：显示"加载中..."文字，等待 planets_initialized
- **4K/超宽屏**：星图区域用 `Control` 居中锚定，留黑边

---

## Accessibility Notes

- 势力区分：颜色 + 位置 + 文字（不依赖单一颜色判断）
- 点击区域：星球命中半径为圆心 ±24px — 足够大，不需要精确点击
- 键盘替代：完整回合循环可通过键盘完成（Space 结束回合 / Esc 取消）

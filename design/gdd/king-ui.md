# 国王 UI (King UI)

> **Status**: In Design
> **Author**: Claude + 老大
> **Last Updated**: 2026-05-31
> **Implements Pillar**: 🕯️ 代际接力 (倒计时是核心压力)
> **Architecture**: [ADR-0008](../docs/architecture/adr/adr-0008-king-lifespan-model.md)

## Overview

国王 UI 显示当前国王的名字、天赋标签、剩余行动次数（寿命倒计时）、代际数。它是玩家时刻可见的"倒计时器"——提醒玩家时间有限，每次结束回合都在消耗。国王去世时展示去世/继位信息。

## Player Fantasy

玩家在星图左上角看到一个小面板："第 1 代：征服者 阿尔萨斯，剩余 23 回合"。数字每回合减少 1。降到 3 时变红——国王快不行了。降到 0 时，一个简洁的提示框弹出："阿尔萨斯 驾崩，享年 30 回合。继承人 伊莎贝尔 继位。" 玩家点"继续"，倒计时重置为 30，代际变为 2。

这就是 Pillar 1（代际接力）和 Pillar 4（自创史诗）的 UI 化身。

## Detailed Rules

### 规则 1: 常驻面板

国王 UI 固定在屏幕左上角，始终可见：

```
┌──────────────────────────┐
│ 👑 第 1 代                │  ← 代际数
│ 征服者 阿尔萨斯           │  ← 天赋 + 名字
│ ████████████░░░░ 23/30   │  ← 寿命条 + 剩余/总量
└──────────────────────────┘
```

### 规则 2: 寿命条

- 长度 = `(lifespan - actions_used) / lifespan`
- 颜色：绿色 (>50%) → 黄色 (30-50%) → 橙色 (10-30%) → 红色 (<10%)
- `< 3` 时寿命条闪烁（警告）

### 规则 3: 死亡/继位弹窗

国王去世时弹出模态窗口：

```
┌──────────────────────────────────┐
│                                  │
│  阿尔萨斯 驾崩                    │
│  第 1 代国王，享年 30 回合         │
│                                  │
│  继承人：伊莎贝尔（囤积者）        │
│  第 2 代，剩余 30 回合            │
│                                  │
│         [ 继续 ]                 │
└──────────────────────────────────┘
```

点击"继续" → 关闭弹窗 → GameState 恢复 PLAYING → 国王 UI 刷新为新国王。

### 规则 4: 天赋标签

- CONQUEROR → "征服者" (红色标签)
- RESEARCHER → "科研者" (蓝色标签)
- HOARDER → "囤积者" (金色标签)
- DIPLOMAT → "外交者" (绿色标签)

MVP 天赋只是标签，无机制效果——UI 上也明确标注"（标签模式）"或直接不标。

### 规则 5: 事件订阅

| 信号 | 响应 |
|------|------|
| `action_consumed(remaining)` | 更新寿命条和数字 |
| `king_died(king)` | 弹出死亡/继位弹窗 |
| `king_succeeded(old, new)` | 刷新国王信息 |
| `game_state_changed` | PAUSED/TITLE 时隐藏面板 |

### States and Transitions

```
PLAYING 状态:
  显示面板（代际+名字+寿命条）

PAUSED 状态 (国王去世):
  面板隐藏 → 弹窗显示 → 玩家点继续 → 弹窗关闭 → 面板恢复

TITLE / VICTORY / DEFEAT:
  面板隐藏
```

## Formulas

不适用。

## Edge Cases

- **初始国王 lifespan=1**: 寿命条 1/1→0/1，立即去世——若配置错误会第一回合结束就弹窗
- **快速双次去世**: lifespan=1 时连续去世——弹窗会快速弹出两次。缓解：MVP lifespan 至少 5
- **名字过长**: 最多显示 6 个中文字符，超出截断+"..."

## Dependencies

| 系统 | 依赖类型 | 依赖内容 |
|------|---------|---------|
| 国王系统 | Hard | `current_king` 全部字段 |
| 事件总线 | Hard | 订阅 3 个国王信号 + `game_state_changed` |

## Tuning Knobs

| Knob | 说明 |
|------|------|
| 面板位置 | 左上角 |
| 寿命条颜色阈值 | 绿>50% 黄30-50% 橙10-30% 红<10% |
| 警告阈值 | remaining <= 3 闪烁 |

## Acceptance Criteria

- **GIVEN** 游戏开始，**WHEN** 国王 UI 渲染，**THEN** 显示"第 1 代：XX 天赋 名字，剩余 30/30"
- **GIVEN** `action_consumed(25)` 信号，**WHEN** UI 刷新，**THEN** 显示"25/30"，寿命条 5/6 长度
- **GIVEN** 剩余 <= 3，**WHEN** UI 刷新，**THEN** 寿命条红色闪烁
- **GIVEN** `king_died` 信号，**WHEN** UI 响应，**THEN** 弹出死亡/继位弹窗
- **GIVEN** 弹窗显示 + 点击"继续"，**WHEN** 完成，**THEN** 弹窗关闭，面板显示新国王信息（第 2 代，30/30）
- **GIVEN** TITLE 状态，**WHEN** 渲染 UI，**THEN** 国王面板隐藏
